# Arduino Servo Motor Interfacing

## Overview
This project demonstrates Servo Motor interfacing with Arduino UNO using Tinkercad.

## Components Used
- Arduino UNO
- Servo Motor (SG90)
- Jumper Wires

## Connections

| Servo Pin | Arduino Pin |
|------------|------------|
| GND | GND |
| VCC | 5V |
| Signal | D7 |

## Working
The servo motor rotates between:
- 0°
- 90°
- 180°

and repeats continuously.
